# 思源黑体字体引入方案

## 当前方案：Google Fonts CDN（推荐）

已在 `public/index.html` 中添加了 Google Fonts CDN 链接。

## 备选方案

### 方案A：使用 jsDelivr CDN（国内可用）

如果无法访问 Google Fonts，可以替换为：

```html
<!-- 使用 jsDelivr CDN -->
<link href="https://cdn.jsdelivr.net/npm/@fontsource/noto-sans-sc@latest/index.css" rel="stylesheet">
```

### 方案B：使用 75CDN（国内CDN）

```html
<!-- 使用 75CDN -->
<link href="https://cdn.bootcdn.net/ajax/libs/fontsource-noto-sans-sc/5.0.0/index.css" rel="stylesheet">
```

### 方案C：下载字体文件到本地

1. 从 [Google Fonts](https://fonts.google.com/noto/specimen/Noto+Sans+SC) 或 [Adobe Fonts](https://fonts.adobe.com/fonts/source-han-sans-simplified-chinese) 下载思源黑体字体文件

2. 将字体文件放到 `public/fonts/` 目录下

3. 在 `src/assets/styles/` 目录下创建 `fonts.scss` 文件：

```scss
@font-face {
  font-family: "Noto Sans SC";
  src: url('../../public/fonts/NotoSansSC-Regular.woff2') format('woff2'),
       url('../../public/fonts/NotoSansSC-Regular.woff') format('woff');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: "Noto Sans SC";
  src: url('../../public/fonts/NotoSansSC-Medium.woff2') format('woff2'),
       url('../../public/fonts/NotoSansSC-Medium.woff') format('woff');
  font-weight: 500;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: "Noto Sans SC";
  src: url('../../public/fonts/NotoSansSC-Bold.woff2') format('woff2'),
       url('../../public/fonts/NotoSansSC-Bold.woff') format('woff');
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}
```

4. 在 `src/assets/styles/index.scss` 中引入：

```scss
@import './fonts.scss';
```

5. 从 `index.html` 中移除 Google Fonts 链接
